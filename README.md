# To Do List with User Accounts

A MERN full stack to-do list that lets users register, log in, and manage personal tasks.

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js

## Features
- User registration with password hashing
- A secure login using JWT Authentication
- Access and Refresh token implementation
- Create, read, and manage custom task headings
- Add tasks under specific headings
